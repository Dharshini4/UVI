instance-n; 1 Senses

Sense Number 1: representative

Commentary: A single occurrence or representation of a larger category.

Examples:
This is an instance of what I'm talking about.
Let's look at riding a bus, for instance.
Instances of bad judgment can be found anywhere.
This patient provides a typical instance of the syndrome.
An individual object of a class is called an "instance" in programming terms.
