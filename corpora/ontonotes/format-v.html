Sense Number 1: Arrange text, data, or other objects in a certain configuration or to certain specifications.

Examples:
We have to format the data before we can run the parsing algorithm.
Format this memo so it can be printed out.
We have to format materials in XML.

Sense Number 2: Divide a computer disk into marked sectors so that it may store data, consequently erasing old data on the disk.

Examples:
Please format this disk before entering the data.
It used to be that you had format disks before using them on your
computer; now you can buy them preformatted.
I need to format this disk for Windows.
