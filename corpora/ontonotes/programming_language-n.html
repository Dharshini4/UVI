programming_language-n; 1 Senses

Sense Number 1: any formal language designed to produce code that runs on a computer

Commentary: PROGRAMMING_LANGUAGE[+language][+mathematical][+for_generation][+of_code][+computer_readable]

Examples:
Lisp is a high-level programming language that has been used to explore approaches to artificial intelligence on a computer.
Forth is a low-level, simple programming language that is useful for driving devices such as motor-driven telescope mounts.
