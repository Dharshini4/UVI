memory-n; 3 Senses

Sense Number 1: Something that is remembered.

Commentary: MEMORY[+remembrance]

Examples:
Search as he would, the memory was lost.
I have great memories of childhood.

Mappings:
WordNet 3.0 Sense Numbers: 1, 2

Sense Number 2: The cognitive processes and capacity to remember.

Commentary: MEMORY[+cognition]

Examples:

He had a better memory when he was younger.

Mappings:
WordNet 3.0 Sense Numbers: 2, 3, 5

Sense Number 3: An electronic storage device.

Commentary: MEMORY[+electronic]

Examples:

Mappings:
WordNet 3.0 Sense Numbers: 4
