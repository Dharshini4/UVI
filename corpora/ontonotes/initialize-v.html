initialize-v; 1 Senses

Sense Number 1: Assign a start value to a computer program or computer disk, such that it begins to operate.

Examples:
The machine could not initialize the program because there wasn't
enough memory.
Initialize the disk drive that you just installed.
Setup cannot properly initialize the disk that you inserted.
